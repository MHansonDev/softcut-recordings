Behringer Neutron

The perfect beginner's tool for learning Modular Synthesis

The Neutron was the hardware synth I had been researching for weeks before finally pulling the trigger. Featuring two oscillators, a Bucket Brigade Delay, and 56 patch points, this synth had more than enough features to help me learn the inner workings of a true analog synthesizer. It can be tedious to use at first, as each oscillator needs to be tuned manually, and an inexperienced user might find the behavior a bit erratic. Though, once I got the hang of controlling each "module", I found it was a wild tool for sound design.
