Open Stage Control

OSC / Midi Touchpad Control

Open Stage Control gave me to opportunity to try out touch based UI / Midi Controller design with minimal setup and configuration. The main reason I chose Open Stage Control over one of the more popular OSC applications is that it's completely free of charge. Being a client/server based editor, once the controller is configured on the desktop, a browser can be used to mirror the controller on a touch based device to sent Midi/CC/OSC messages to a set of devices. While I don't yet have a practical way to implement this into the setup, I still found it fun to trigger samples in Cubase while sitting on the couch in the living room.
